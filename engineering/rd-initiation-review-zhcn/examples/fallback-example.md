# 降级示例

若结构化专利/论文检索不可用，本技能仍应按以下方式继续执行：

1. 从提案材料出发，仔细提取主张。
2. 在 `method_decisions.md` 中记录最高置信度的结构化来源不可用。
3. 尽可能使用任何可用的学术数据库、官方申报文件或领域专项来源。
4. 使用 `Exa` 进行广域公开网络发现。
5. 若可用，使用 `Tavily` 或 `Brave` 进行补充发现。
6. 使用通用 `web` 检索作为降级方案。
7. 使用 `Jina` 或其他已知 URL 阅读器，读取关键的官方页面或 PDF。

最终报告应明确说明，新颖性和重叠结论是在降级证据路径下得出的。
