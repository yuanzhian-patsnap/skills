# 来源路由

使用实际可用的最高置信度通道。

## 默认优先顺序

1. 结构化专利和论文检索
2. 可用时优先使用领域专属学术数据库
3. `Exa`：用于公开网络发现和官方页面
4. `Tavily`：可用且有效时使用
5. `Brave`：作为额外降级选项
6. `web`：通用搜索降级
7. `Jina` 或其他已知 URL 阅读器：发现后用于阅读页面或 PDF

## 按信息需求路由

| 需求 | 首选通道 | 降级通道 |
| --- | --- | --- |
| 路线定义与候选证据 | 结构化检索 | `Exa` + `web` |
| 前沿论文与 DOI 支持 | 结构化检索或学术数据库 | `Exa` + `web` |
| 标准、发布及官方公开状态 | `Exa` | `web`，再用 `Jina` |
| 详细页面或 PDF 阅读 | 发现后用 `Jina` | 任何等效的已知 URL 阅读器 |

## 降级规则

若结构化检索不可用：

- 在 `method_decisions.md` 中说明
- 使用领域专属来源、`Exa`、`Tavily`、`Brave`、`web` 和 `Jina`
- 降低路线广度、新颖性声明和前沿完整性的置信度

若路线证据明显偏薄，不得将降级运行呈现为详尽分析。
