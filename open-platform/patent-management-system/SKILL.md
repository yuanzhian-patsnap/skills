# 专利管理系统构建 Skill

## 定位与背景

本 Skill 封装了为**杭州恩和生物科技有限公司**构建 AI 赋能专利管理系统的完整流程、提示词模板、数据结构和功能模块设计。可复用于任何合成生物学、生命科学企业的专利管理系统建设。

核心依据：
- 《专利明镜——AI赋能知识产权管理方案》PPT（智慧芽官方能力图谱）
- inno.pptx（真实客户需求调研，含辉大基因等案例）
- 恩和生物实际专利数据（83条，含同族，通过 PatSnap API 拉取）

## 核心工作流程

### Step 1：拉取企业专利数据
```
调用工具：patsnap_search
查询参数：assignees=["杭州恩和生物科技有限公司", "恩和生物"]
返回字段：pn, title, patentId, legal_status, application_date, ipc
去重逻辑：按 APNO_FACET 去重，取最新公告号
目标数量：全量（恩和生物约83条含同族）
```

### Step 2：识别竞争对手
三维识别逻辑：
1. **IPC相同**：提取恩和生物主要IPC（C12N/C12P/A61K/C07K），搜索同IPC企业
2. **引用关系**：被恩和引用过的 + 引用过恩和的专利持有人
3. **同技术方向**：关键词（合成生物学/虾青素/胶原蛋白/辅酶Q10/维生素K2）

全球竞对参考列表（已验证真实存在）：
- 国内：巨子生物、锦波生物、江苏创健、华恒生物、凯赛生物、中科院天津工生所
- 国际：DSM-Firmenich、Evonik、Ginkgo Bioworks、Amyris、Genomatica、Evolva

### Step 3：生成 HTML 系统
输出要求：
- 单文件 HTML，UTF-8编码，无外部依赖（Chart.js 用 CDN）
- 所有专利链接格式：`https://eureka.zhihuiya.com/view/#/fullText?patentId={patentId}`
- 支持左侧导航切页，8个模块全部可用
- 一键同步按钮：动画进度弹窗，6步同步流程

## 8个核心模块设计规范

### 模块1：工作台（Dashboard）
- 统计卡片：总量/有效/审查中/PCT国际
- 图表1：法律状态饼图（Chart.js Doughnut）
- 图表2：专利类型饼图（发明/实用新型/PCT）
- 图表3：年度申请趋势柱状图
- 图表4：技术方向分布饼图
- 待办清单：年费到期提醒 + 审查意见待回复

### 模块2：专利资产库
- 搜索框 + 状态筛选 + 类型筛选
- 分页表格（每页10条）
- 列：序号/公开号（可点击跳转智慧芽）/标题/申请日/状态/类型/IPC/操作
- 状态标签颜色：active=绿色, pending=黄色, inactive=红色

### 模块3：费用管理
- 到期预警列表（按时间排序）
- 每条：专利号/名称/年费金额/到期日/「已缴费」按钮
- 已缴费状态切换（绿色标记）

### 模块4：代理所分析
- 饼图：各代理所专利数量分布
- 柱图：各代理所授权率对比
- 明细表：代理所名称/专利数/授权数/授权率/主要方向

恩和生物已知代理所（基于专利特征推断）：
- 浙江杭知桥知识产权代理有限公司（主要）
- 杭州求是专利事务所
- 北京集佳知识产权代理有限公司（PCT方向）

### 模块5：竞企追踪
- 竞对卡片网格布局
- 每张卡片：公司名/匹配原因（IPC/引用/同技术）/风险等级条/专利数/一键FTO按钮
- 风险等级：高=红色, 中=橙色, 低=绿色
- 点击「FTO分析」跳转到FTO模块并自动填入企业名

### 模块6：FTO 风险分析
- 左侧表单：
  - 企业名称（datalist自由输入，内置20家候选）
  - 分析产品/技术特征（textarea）
  - 技术领域（select）
  - 「调用智慧芽AI·开始分析」按钮
- 右侧结果区：
  - 综合风险等级（高/中/低徽章）
  - 命中专利列表（含链接）
  - 具体风险说明
  - 规避建议

### 模块7：专利价值评估
- TOP10排名列表
- 每条：排名圆圈/专利号+标题/综合评分+T技术/L法律/B商业三维分值
- 「一键更新」按钮（调用智慧芽价值评估API）

### 模块8：AI 查新检索
- 左侧输入区：
  - 大文本框（技术方案描述）
  - 合成生物学样例按钮（一键填充）：
    - 虾青素生物合成
    - 重组人胶原蛋白
    - RecET基因编辑系统
    - 维生素K2(MK-7)发酵
    - 辅酶Q10工程菌
  - 「开始查新」按钮
- 右侧结果区：相似专利列表 + 新颖性评估

## 避坑指南（来自实际构建经验）

1. **数据量丢失问题**：每次迭代必须从上一版本读取完整专利列表再修改，禁止重新截断
2. **链接失效问题**：专利链接必须用 `https://eureka.zhihuiya.com/view/#/fullText?patentId={真实patentId}` 格式，patentId 来自 API 返回的 UUID
3. **编码问题**：HTML文件写入时使用 Python files 工具直接写入，不经过 exec/GBK 编码环境
4. **图表不渲染**：Chart.js 必须用 CDN `https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js`，且 canvas 必须在页面 show 后才初始化
5. **竞对数据**：不能用假数据，必须通过 patsnap_search 搜索 IPC 相同企业获取真实竞对列表

## 项目背景（需求文档摘要）

**企业画像**：杭州恩和生物科技有限公司，成立于2021年，专注合成生物学，产品覆盖虾青素、辅酶Q10、维生素K2、重组蛋白（胶原蛋白/弹性蛋白/纤连蛋白/角蛋白）等，已申请专利83件（含同族），PCT国际专利2件。

**核心痛点**：
1. 专利到期日分散管理，人工维护容易遗漏年费缴纳
2. 竞对专利监控靠人工检索，效率低、覆盖不全
3. 查新委外费用高（年均数十万），周期长（3-7天）
4. FTO 风险评估缺乏系统化工具，新产品上市前风险暴露

**成功标准**：
- MVP（6-8周）：专利资产库+费用预警+代理所分析+竞企追踪
- Phase 2（3-4月）：FTO分析+专利价值评估
- Phase 3（6月+）：AI查新检索
