# 专利技术全景分析报告结构指南

## 报告五大模块

基于专利分析维度框架，报告必须包含以下5个模块。每个模块有固定的分析维度和可视化要求。

### 模块01：技术概况分析
分析维度：
- **关键指标卡片**：专利总数、年申请峰值、授权率、中国/海外占比、核心申请人数量
- **专利申请趋势**：按申请年的折线图（近10年）
- **法律状态分析**：授权/实质审查/公开/PCT的饼图或环形图
- **全球地域分布**：受理局/国家的柱状图，中美欧趋势对比
- **技术构成分布**：按技术分支的饼图或环形图
- **主要申请人排名**：Top15申请人的横向柱状图

### 模块02：重点技术分支分析
分析维度：
- **各分支专利趋势**：多条折线图（2020-2025），展示各技术分支的年度变化
- **技术热点卡片**：增长率>100%的分支，标注增长率和核心方向
- **技术空白点卡片**：专利量少、增长慢或负增长的分支
- **技术分解表**：参考用户提供的技术拆解Excel，构建一级/二级分支表格，含专利数量、主要申请人、活跃度
- **研发路线演进**：时间线形式，通常分4个阶段（基础期→突破期→驱动期→产业化期）

### 模块03：重点竞争对手分析
分析维度：
- **竞争格局总览**：雷达图，对比Top5-8申请人的多技术维度布局
- **三梯队划分**：国际龙头（雷神/GE/罗罗等）、国内航发系、高校/研究机构
- **竞争对手卡片**：每梯队包含申请人名称、专利数、核心方向、竞争策略
- **技术策略对比矩阵**：表格形式，含核心布局技术、专利策略、技术路线、优劣势

### 模块04：专利风险分析
分析维度：
- **涉诉专利风险**：高被引基础专利（被引>20次），多国同族专利壁垒
- **重点技术风险**：按技术领域标注风险等级（高/中高/中），识别专利重叠区域
- **风险缓解建议**：FTO分析优先、规避设计路径、专利联盟策略
- **重点专利解读表格**：公开号、申请人、技术方向、被引次数、风险等级、技术要点

### 模块05：企业专利布局建议
分析维度：
- **自有专利梳理与差距分析**：优势领域（持续加强）+ 差距领域（重点补强）
- **竞争对手对标布局**：表格形式，对标对象、技术方向、布局策略、预期效果、优先级（P0/P1/P2）
- **重点分支布局策略**：4大方向建议，每个方向含策略说明
- **风险规避建议**：FTO必须领域、规避设计方向、专利合作建议、出海布局建议
- **技术吸引力-专利密度矩阵**：散点图，X轴专利密度Y轴增长率，四象限矩阵

## 数据列名映射（常见变体）

| 标准列名 | 常见变体 |
|---------|---------|
| 公开(公告)号 | 公开号, 公告号, 专利号, Publication Number |
| 标题 | 专利标题, 发明名称, Title |
| 法律状态/事件 | 法律状态, 案件状态, Legal Status |
| [标]当前申请(专利权)人 | 申请人, 专利权人, Applicant, Assignee |
| 申请日 | 申请日期, Filing Date |
| 公开(公告)日 | 公开日, 公告日, Publication Date |
| 受理局 | 国家/地区, Country, 申请国 |
| 被引用专利数量 | 被引用次数, Cited Count |
| 简单同族国家/地区 | 同族国家, Family Countries |

## 申请人名称清理规则

需要统一名称变体（中英文、大小写、缩写等）：
- 雷神科技/RTX/Raytheon → 雷神科技
- 通用电气/GE/General Electric → 通用电气
- 罗尔斯·罗伊斯/Rolls-Royce/罗尔斯 罗伊斯/劳斯莱斯 → 罗尔斯·罗伊斯
- 赛峰/Safran/赛峰集团 → 赛峰集团
- 中国航发XX研究所 → 中国航发+简称（动研所/涡轮院/动力所/商发/航材院）
- 各大学 → 大学全称简称
