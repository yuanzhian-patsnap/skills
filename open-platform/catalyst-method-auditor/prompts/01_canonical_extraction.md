# 方案要素结构化抽取提示词

仅在脚本无法直接读取结构时使用。抽取结果必须忠于原文，不得补写。

需要抽取：

- 材料类型：制备步骤型方案 / 制备与评价完整方案 / 研发设想 / 论文方法片段 / 专利实施例 / 材料不足；
- 催化剂体系和样品名称；
- 制备步骤：试剂、用量、溶剂、混合顺序、温度、时间、气氛、洗涤、干燥、煅烧、还原；
- 样品设计：目标样、对照样、空白样、变量差异；
- 评价计划：反应、反应器、温度、压力、空速/流量、催化剂用量、分析方法；
- 表征计划：XRD、XPS、TEM、BET、ICP、TPR/TPD 等；
- 性能或机理主张。

缺失信息必须写“未明确”。
