# Competitive Patent Landscape — 竞品最新专利布局分析

## 适用场景

客户高层（总裁/VP/CXO）需要了解**竞品在押注哪些新产品方向**，而非做专利侵权分析。
核心问题：竞品在保护什么 → 我的机会在哪里 → 现在进还是晚了？

典型使用场景：
- 某行业客户希望通过竞品专利布局，洞察对手未来 1-3 年的新品策略
- 向客户总裁做竞争态势汇报，需要有数据支撑的专业报告
- 跨市场竞品对比（如同一品牌在美国 vs 欧洲的布局差异）

## 输入参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `industry` | 行业/品类名称 | 面巾纸、手帕纸、固态电池 |
| `brands` | 竞品品牌列表（含市场标注） | 金佰利(美国/欧洲)、宝洁(美国)、ESSITY(欧洲) |
| `client_name` | 客户名称（用于报告标题和维达视角洞察） | 维达纸业 |
| `report_title` | 报告标题（可选，默认自动生成） | 全球主要市场头部品牌新品战略洞察 |
| `brand_color` | 客户品牌主色调（十六进制，可选） | #1a5ca8 |
| `years` | 分析年限（默认5年） | 5 |

## 输出

- 完整 HTML 报告（浅色品牌风格，Chart.js 图表，锚点导航）
- 保存路径：`@session/output/<client_name>_patent_landscape.html`

## 报告五章结构

### 第一章：全局一览 + 分析结论速览（前置）
- **分析结论速览模块**（置于图表最前）：
  - 三大核心技术路线趋势（卡片形式）
  - 各品牌布局重点一览表（含窗口判断：封锁级/12-18月窗口/可进入/跟踪观察）
  - 一句话总结引用块
- 竞争格局气泡图（x=专利数量，y=技术宽度，气泡=引用量）
- 各品牌专利数量横向柱状图

### 第二章：分品牌深度分析
每个品牌一张"技术战略卡"，包含：
- 专利簇分布（条形图 + 占比）
- 三大核心押注方向（具体专利号 + 技术要点）
- 对客户的洞察（机会 or 风险）

### 第三章：核心专利全文解读
- 每个品牌选 2-3 件代表性专利 fetch 全文
- 解读：核心技术主张 + 全球布局路径（优先权→PCT→各市场）
- 品牌全球布局战略对比总结表

### 第四章：全球布局战略对比
- 各品牌布局路径对比（优先权→PCT→市场进入）
- 地理热力矩阵（品牌 × 国家，颜色=专利密度）
- 特殊跨市场对比（如金佰利美国 vs 欧洲）

### 第五章：战略结论与机会窗口（压轴）
- 三个可进入的技术空白（卡片形式）
- 时间窗口判断表（绿/黄/红三色标注）
- 对客户的行动建议（3-5条）

## 工作流程

```
Step 1: 理解输入
  - 解析品牌列表和市场范围
  - 确认客户名称和品牌色调

Step 2: 专利检索（每个品牌独立检索）
  - 策略：keyword + filter（按申请人名称过滤）
  - 时间范围：近5年
  - topk：每品牌 50-100 件
  - 同时检索：semantic query（技术语义）+ keyword（品牌名）

Step 3: 数据汇总
  - 按品牌统计专利数量
  - 按 IPC 大组聚类为技术路线（6条标准路线）
  - 按 jurisdiction 统计地理布局
  - 按引用量排序，选 Top 3 候选精读专利

Step 4: 核心专利精读（每品牌 2-3 件）
  - fetch 全文 markdown
  - 提取：核心技术主张、权利要求要点、优先权路径

Step 5: 生成分析结论
  - 总结技术路线趋势（3条）
  - 为每个品牌写专利簇分析和新品策略解读
  - 识别客户的机会窗口（技术空白 + 时间窗口判断）

Step 6: 生成 HTML 报告
  - 五章结构，浅色品牌风格
  - Chart.js 图表（气泡图/雷达图/热力矩阵/柱状图）
  - 锚点导航，响应式布局
  - 保存至 @session/output/
```

## 设计规范

### 视觉风格
- 底色：`#f5f7fa`（浅灰白）
- 主色：使用客户品牌色（默认 `#1a5ca8`）
- 卡片背景：白色 `#ffffff`
- 绝对禁止：黑底/深色背景主题
- 字体：PingFang SC / Microsoft YaHei

### 图表规范
| 图表 | 用途 | 说明 |
|------|------|------|
| 气泡图 | 竞争格局 | x=专利量，y=技术广度，气泡=引用量 |
| 横向柱状图 | 品牌专利数量对比 | 降序排列 |
| 雷达图 | 各品牌技术路线覆盖 | 6维度 |
| 堆叠面积图 | 技术路线趋势 | 近5年 |
| 热力矩阵 | 地理布局 | 品牌×国家 |

### 结论写作规范
- ❌ 禁止：「金佰利在美国有47件专利，在欧洲有23件」
- ✅ 要求：「金佰利美国布局是欧洲2倍，重点在工艺保护，说明欧洲以产品防御为主」
- 每张图表下方必须有**1句话核心结论**
- 每个品牌分析必须有**对客户的洞察**（机会或风险）

## 注意事项

1. **视角是新品策略洞察，不是侵权分析**——客户目的是看竞品押注哪些方向，推断其新品策略
2. **日文专利补充**：日本品牌需同时使用英文+中文关键词检索，或按申请人精确过滤
3. **样本说明**：检索返回为 topk 样本，排名为样本内相对排名，需在报告中注明
4. **跨市场对比**：若同一品牌在多个市场有布局（如金佰利），必须专门做差异对比分析
5. **窗口判断逻辑**：
   - 🔴 封锁级：竞品已建立深厚壁垒，规避成本高
   - 🟡 12-18月窗口：竞品领先但参数空间未满
   - 🟢 可进入：空白较大，建议结合高校联合申请
   - ⬜ 跟踪观察：新兴信号，暂不需要行动
