# 仿药速探 V1 Scope

V1 is intentionally narrow:

- China / CN.
- Small-molecule chemical drugs.
- Crystal-form / polymorph patents.
- Phase 3 and later drug evidence.

Rationale:

- Crystal-form patent screening is most meaningful for small-molecule chemical drugs.
- For V1, a narrow scope reduces false positives and makes the output easier to explain.
- Compound patent expiry is still shown because it helps judge whether the crystal-form patent is a remaining barrier, but it should not exclude candidates automatically.

Report and UI sections to preserve:

- Main-table candidates with complete evidence.
- Priority candidate explanation.
- Excluded / not-promoted candidates.
- Data notes and limitations.
- Sources.

Value message:

仿药速探 V1 turns repeated quarterly manual searches into a standardized, reusable screening workflow for early generic-drug opportunity discovery. It helps reduce repetitive work, avoid missed candidates, preserve evidence, and let analysts focus on product judgment.
