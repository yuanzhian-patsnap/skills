# Korea Patent Translation Reference

## Target Language

Use Korean. Format for Korean patent application drafting.

## Drafting Focus

- Use Korean patent drafting conventions rather than Chinese-literal wording.
- Keep claim limitations, technical relationships, and execution subjects clear.
- Maintain consistent translations for "부", "모듈", "유닛", "장치", "수단", and "단계".
- Avoid adding unsupported technical effects, parameters, or implementation details.
- Preserve support between claims and detailed description.

## Recommended Section Order

```text
발명의 명칭
기술분야
배경기술
발명의 내용
도면의 간단한 설명
발명을 실시하기 위한 구체적인 내용
청구범위
요약서
도면
서열목록（해당하는 경우）
```

## Claim Style Notes

- Use "포함하는" for open-ended "包括/包含" when appropriate.
- Keep "하나 이상", "복수의", "임의의", and "및/또는" scope terms accurate.
- Avoid strict sequence wording unless source text or technical logic requires it.
- Keep device, method, system, and storage-medium claim categories distinct.
- Translate connection/coupling terms according to structural, electrical, communication, or logical relation.

## QA Focus

- Check that Korean syntax does not shift modifiers to the wrong technical element.
- Check that optional embodiments remain optional.
- Check reference signs, units, formulas, and parameters against the Chinese source.
