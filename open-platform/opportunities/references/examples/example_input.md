# 示例输入

## 示例 1：细分技术方向

**输入：**
```
请分析「钠离子电池硬碳负极预钠化技术」的专利机会
```

**预期 Skill 行为：**
1. 识别为细分技术方向 ✅
2. 构建检索式：`TACD:(sodium ion battery OR 钠离子电池) AND TACD:(hard carbon OR 硬碳 OR 预钠化 OR pre-sodiation)`
3. 调用 trends MCP 获取全量年度趋势（2015至今）
4. 调用 applicant_ranking MCP 获取全量申请人排名
5. 调用 priority_country MCP 获取地域分布
6. 调用 simple_legal_status MCP 获取法律状态
7. 对 4-6 个子技术方向分别检索获取数量
8. 搜索代表专利（TopK50）作为 evidence_sample
9. 生成 10 文件完整报告

## 示例 3：需要拒绝的宏观输入

**输入：**
```
请分析「新能源」的专利机会
```

**预期 Skill 行为：**
1. 识别为过于宏观 ❌
2. 不生成报告
3. 回复：
```
您的输入「新能源」范围过于宏观，无法生成精准的专利机会评估报告。

请将技术方向细化，例如：
- 钠离子电池硬碳负极技术
- 钙钛矿太阳能电池封装材料  
- 全固态电池硫化物电解质
- 风电叶片疲劳寿命预测方法
- 储能系统热管理技术

请提供您具体关注的细分技术方向或技术方案。
```
