# Scoring Prompt — 综合评分

## 任务
基于所有收集到的全量统计数据和分析结论，生成综合推荐评分。

## 评分维度（满分 100）

### 维度 1：专利增长趋势（0-20 分）
| 得分 | 条件 |
|------|------|
| 17-20 | 近五年 CAGR > 25%，近三年仍在加速 |
| 13-16 | 近五年 CAGR 15-25%，增长稳定 |
| 9-12 | 近五年 CAGR 5-15%，增长温和 |
| 5-8 | 近五年 CAGR < 5%，增长停滞 |
| 0-4 | 近三年出现下降 |

数据来源：full_scope_metrics.trend_yearly_application（必须是全量数据）

### 维度 2：申请人活跃度（0-15 分）
| 得分 | 条件 |
|------|------|
| 12-15 | 活跃申请人 > 100，近三年新进入者多 |
| 8-11 | 活跃申请人 50-100，格局相对稳定 |
| 4-7 | 活跃申请人 < 50，格局固化 |
| 0-3 | 极度集中，新进入空间极小 |

数据来源：full_scope_metrics.top_applicants 和申请人总数

### 维度 3：专利质量/影响力（0-15 分）
| 得分 | 条件 |
|------|------|
| 12-15 | 高被引专利多，大同族多，有效率 > 60% |
| 8-11 | 有一定被引专利，有效率 40-60% |
| 4-7 | 质量一般，有效率 < 40% |
| 0-3 | 大量低质量专利，有效率极低 |

数据来源：most_cited MCP + legal_status MCP

### 维度 4：子技术机会空间（0-20 分）
| 得分 | 条件 |
|------|------|
| 17-20 | ≥ 3 个子技术方向"推荐进入"，空白点多 |
| 13-16 | 1-2 个子技术方向"推荐进入" |
| 8-12 | 子技术方向主要为"谨慎进入" |
| 0-7 | 所有方向均"暂不建议进入" |

数据来源：subfield_analysis 评估结果

### 维度 5：竞争风险（反向，0-15 分）
| 得分 | 条件（得分越高说明竞争越低） |
|------|------|
| 12-15 | CR5 < 30%，高度分散，新进入易 |
| 8-11 | CR5 30-50%，中度分散 |
| 4-7 | CR5 50-70%，中度集中 |
| 0-3 | CR5 > 70%，高度集中，进入壁垒高 |

### 维度 6：全球布局/商业化信号（0-15 分）
| 得分 | 条件 |
|------|------|
| 12-15 | PCT 申请多，美欧日韩均有布局，有商业化信号 |
| 8-11 | PCT 申请中等，国际布局部分 |
| 4-7 | 主要集中在单一国家，国际化弱 |
| 0-3 | 几乎无国际布局 |

## 综合结论映射

| 综合分 | 研发进入建议 | 投资吸引力 | 证据置信度 |
|--------|-------------|------------|------------|
| 75-100 | 建议进入 | 高 | 视数据质量 |
| 50-74 | 谨慎进入 | 中 | 视数据质量 |
| 0-49 | 暂不建议进入 | 低 | 视数据质量 |

证据置信度评定：
- **高**：所有关键统计均来自全量 MCP 数据，无缺口
- **中**：部分统计来自分桶检索，有少量推断
- **低**：有多项统计缺失，结论依赖推断
