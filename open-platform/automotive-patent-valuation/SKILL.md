# 汽车行业高价值专利评估 Skill

## 适用场景

- 对任意汽车企业（国内外）进行高价值专利筛选与评分
- 与主流车企（比亚迪/东风/丰田/现代/特斯拉等）专利竞争力对标
- 结合国资委14项高价值专利标签进行政策合规性评估
- 输出专利战略建议：PCT布局、获奖申报、SEP认定、侵权诉讼行动

---

## 核心模型：14维三大维度高价值专利评估模型 v4.0

### 大维度权重架构

| 大维度 | 权重 | 核心逻辑 |
|-------|-----|---------|
| 技术维度 | **40%** | 衡量专利技术先进性、战略价值、全球布局广度与行业影响力 |
| 法律维度 | **30%** | 衡量专利权利稳定性、保护范围质量、抗风险能力与法律执行力 |
| 经济维度 | **30%** | 衡量专利商业变现能力、市场认可度与标准卡位价值 |

---

### 14维子维度评分标准

#### 技术维度（40%）

**D1 技术成熟度 TRL（全局权重 8.0%）**
- TRL 8-9（量产/商用）= 10分
- TRL 6-7（示范/验证）= 7分
- TRL 4-5（原型/实验室）= 4分
- TRL 1-3（概念/基础研究）= 2分

**D2 技术战略相关性（全局权重 10.0%）**
- 国家卡脖子技术（芯片/固态电池/操作系统等）= 10分
- 全球十大趋势 Top 1-3 = 9分
- 全球十大趋势 Top 4-7 = 7分
- 一般汽车技术方向 = 4分

**D3 同族国际布局（全局权重 8.0%）** ⭐ 归属技术维度
- ≥5个国家/地区有效专利 = 10分
- ≥3个国家有效专利 = 7分
- US + EP 双覆盖 = 6分
- PCT 在审中 = 4分
- 仅 CN = 2分

**D4 引用影响力（全局权重 8.0%）**
- 被引≥20次 = 10分；≥10次 = 8分；≥5次 = 6分；≥3次 = 4分；≥1次 = 2分；0次 = 1分
- 年均增速 >3次/年：额外+1分

**D5 产业生态影响力（全局权重 6.0%）**
- 被下游产业链企业引用 = 10分
- 被直接竞争对手引用 = 8分
- 仅被学术/科研机构引用 = 5分
- 无引用记录 = 1分

---

#### 法律维度（30%）

**D6 法律稳定性（全局权重 5.1%）**
- 有效（active）且同族多国均有效 = 10分（+2加权）
- 有效（active，仅CN）= 10分
- 审查中（pending）= 5分
- 失效（inactive）= 0分

**D7 权利要求保护范围（全局权重 6.0%）** ⭐ 归属法律维度，倒U形曲线
- 1-3项 = 4分（保护范围过窄，易被规避）
- 4-5项 = 6分
- 6-7项 = 8分
- **8-11项 = 10分** ⭐ 最优区间
- 12-15项 = 7分
- 16-20项 = 4分
- >20项 = 2分（过度碎片化）

**D8 无效风险稳定性（全局权重 4.8%）**
- 审查意见0-1次且无无效请求 = 10分
- 审查意见2次 = 7分
- 曾被提无效请求但最终维持有效 = 6分
- 高无效风险（多次被挑战/审查≥3次）= 2分

**D9 标准必要专利认定（全局权重 5.1%）**
- 已正式声明SEP入库（ETSI/CCSA/IEEE/GB/T）= 10分
- SEP声明申报中 = 7分
- 技术方案被标准采纳但未声明 = 5分（潜在SEP，需主动申报）
- 技术方向与标准高度相关 = 3分
- 与标准无关联 = 1分

**D10 发明专利维护年限（全局权重 4.5%）**
- ≥15年 = 10分；≥10年 = 8分；≥7年 = 6分；≥5年 = 4分；≥3年 = 2分；<3年 = 1分
- 实用新型/外观设计 = 0分（本维度不适用）

**D11 侵权诉讼维度（全局权重 4.5%）**
- 原告起诉并胜诉（有判决/和解记录）= 10分
- 原告起诉，案件进行中 = 7分
- 被他方列为规避/无效目标但未构成侵权 = 6分
- 被诉侵权但胜诉维持有效 = 5分
- 无诉讼记录 = 3分（中性分，不惩罚新专利）
- 被诉侵权且败诉/部分无效 = 0分

---

#### 经济维度（30%）

**D12 商业转化信号（全局权重 9.0%）**
- 有正式许可协议记录 = 10分
- 有专利转让记录 = 8分
- 有明确产品落地（可追溯）= 7分
- 有质押融资记录 = 6分
- 无任何商业转化记录 = 1分

**D13 获奖荣誉等级（全局权重 10.5%）**
- 国家科技进步奖一等/二等 = 10分
- 中国专利金奖（国知局）= 10分
- 省部级科技进步一等奖 / 中国专利银奖 = 8分
- 中国专利优秀奖 / 省部级二等奖 = 6分
- 省部级三等奖 = 5分
- 汽车行业/学会级奖项 = 4分
- **无获奖记录 = 0分**（严格设定，强力激励主动申报）

**D14 标准关联度（全局权重 10.5%）**
- 被GB/T或ISO标准正式引用 = 10分
- 参与国家/行业标准草案编写 = 8分
- 技术方向与现行标准高度相关 = 5分
- 与标准无明确关联 = 1分

---

### 高价值专利认定门槛

**加权总分 ≥ 6.5分** → 认定为高价值专利

附加三项硬性条件（任一不满足则降级为C级临界观察）：
1. D4（引用影响力）≥ 2分
2. D6（法律状态）必须为有效（active）
3. D11（侵权诉讼）≠ 0分（被诉败诉专利不进入高价值名单）

---

### 专利分级体系

| 级别 | 分数段 | 管理策略 |
|-----|-------|---------|
| A级 战略核心 | 8.0-10.0 | 最高保护级别，主动运营，PCT优先布局 |
| B级 高价值认定 | 6.5-7.9 | 维权监控，标准卡位，外围专利扩展 |
| C级 临界观察 | 5.5-6.4 | 6-12个月行动窗口，补强D3/D7/D13或降级 |
| D级 储备专利 | 4.0-5.4 | 定期复评，等待TRL提升/引用积累 |
| E级 低价值/清理 | <4.0 | 考虑放弃续费或组合转让 |

## 标准执行流程

### Step 1：检索目标企业专利
使用 patent.search 工具，参数：
- assignees: [目标企业名称]
- legal_status: active
- date_from: 根据分析需要设定
- sources: patent
- topk: 50（可分批扩大至200）

### Step 2：竞品对标检索
同步检索：比亚迪/东风/丰田/现代/特斯拉等，相同技术方向

### Step 3：逐项评分
对每件专利按D1-D14逐维度评分，优先用工具查询：
- 法律状态 → get_patent_legal_status
- 被引数量 → forward_citation
- 同族数量 → family
- 获奖记录 → award_data
- 许可/转让 → license_data / transfer_data
- 复审无效 → reexamination_invalidation

### Step 4：加权计算
总分 = Σ(各维度原始分 × 全局权重)
技术维度得分 = (D1×0.08 + D2×0.10 + D3×0.08 + D4×0.08 + D5×0.06) / 0.40
法律维度得分 = (D6×0.051 + D7×0.06 + D8×0.048 + D9×0.051 + D10×0.045 + D11×0.045) / 0.30
经济维度得分 = (D12×0.09 + D13×0.105 + D14×0.105) / 0.30

### Step 5：分级输出
按A/B/C/D/E五级输出高价值专利名单，附战略建议

## 输出格式规范

每次分析必须输出以下表格：
1. 检索数据基础总览表
2. 代表专利14维评分明细表
3. 竞品对标评分对比表
4. 国资委14标签匹配评估表
5. 专利分级名单（A/B/C/D/E）
6. 战略行动矩阵（优先级×行动×目标）
