# 雄安国创中心企业及技术专项尽调报告技能

## 概述

本技能面向雄安国创中心招商/投资/落地评审场景，整合外部公开数据与内部业务规则，按七步标准工作流全自动生成专业尽调报告，输出 HTML 格式（可转 PDF）。

## 触发场景

- 用户输入目标企业名称（+社会信用代码、+核心技术方向，可选）
- 明确触发词：企业尽调、技术尽调、尽调报告、IP尽调、雄安尽调、国创中心尽调
- 用户上传或提及已有尽调材料，需要深化分析或补全

## 输入参数

| 参数 | 必选 | 说明 |
|------|------|------|
| 目标企业全称 | ✅ | 用于专利/工商/舆情检索主键 |
| 统一社会信用代码 | 建议填写 | 用于精准工商匹配 |
| 核心技术关键词 | ✅ | 用于专利检索、竞品识别、FTO排查 |
| 尽调场景 | 可选 | 招商落地 / 投资合作 / 技术引进（默认：招商落地） |
| 竞品范围 | 可选 | 指定竞品企业名单，不填则自动识别 |

## 七步工作流

### Step 1 — 自动立项 & 尽调范围界定
- 识别企业所属行业、主营业务、核心技术赛道
- 匹配雄安国创中心尽调场景（招商/投资/落地）
- 输出：尽调范围说明卡（行业、场景、边界清单）

### Step 2 — 搭建报告框架
按固定模板生成九章目录：
1. 执行摘要（一页式）
2. 企业概况
3. 核心技术解析
4. 知识产权尽调
5. 研发能力评估
6. 竞品技术对标
7. 法律合规风险
8. 风险等级汇总
9. 尽调结论与落地建议

### Step 3 — 外部数据检索 & 清洗分析
调用工具：
- `patent.search` / `patent.fetch`：检索企业全量专利，获取法律状态、权属、质押、技术布局
- `paper.search`：检索核心技术领域论文，评估技术深度与学术支撑
- `web.search` + `web_fetch`：抓取工商信息、股权穿透、高管背景、融资历史、负面舆情、涉诉仲裁、行政处罚

### Step 4 — 内部规则融合（基于对话输入模拟）
- 按内置风险打分规则自动对风险点做三级标注：🔴 致命风险 / 🟠 重大风险 / 🟡 一般风险
- 参考内置行业基准库，补充行业技术壁垒与准入门槛说明

### Step 5 — 技术深度分析
- 拆解核心技术原理、技术壁垒、TRL 成熟度（1-9级评估）
- FTO 自由实施风险排查：识别是否落入他人专利保护范围
- 核心专利稳定性评估：权属争议、职务发明风险、转让受让合规性
- 研发团队与投入评估

### Step 6 — 报告整合输出
- 融合所有分析数据，按九章目录填充完整报告
- 生成附件：专利清单表、工商简表、风险清单、竞品对比表
- 提炼执行摘要，高亮重大风险与核心优势

### Step 7 — HTML 报告渲染
- 输出专业深色风格 HTML 报告（含 ECharts 可视化图表）
- 附件表格嵌入报告末尾
- 支持浏览器直接打开并打印为 PDF

## 输出物

| 产物 | 格式 | 说明 |
|------|------|------|
| 完整尽调报告 | HTML | 九章结构，含可视化图表 |
| 执行摘要 | HTML 内嵌 | 一页式，高亮风险 |
| 专利清单 | HTML 表格 | 含法律状态、权属、技术分类 |
| 竞品对比表 | HTML 表格 | 技术路线 × 专利布局 × 优劣势 |
| 风险清单 | HTML 表格 | 三级风险标注，含影响说明 |

## 数据溯源原则

- 所有专利数据标注来源（智慧芽专利数据库）及检索时间
- 所有网络信息标注来源 URL 及抓取时间
- 内部规则打分标注规则版本（默认：国创中心风险评级规则 v1.0）
- 无法核实的信息标注「待核实」，不主观生成结论

## 约束与护栏

1. 全程遵循雄安国创中心尽调业务视角（招商/投资/落地），不偏离园区评审逻辑
2. 所有结论必须依托工具返回的真实数据，禁止主观臆断
3. 风险点强制三级高亮标注，并附简要风险影响说明
4. 每条专利/企业信息/竞品数据保留来源溯源标记
5. 内部敏感数据（园区台账、案例库）以用户上传文件为准，技能不自行生成虚假内部数据

## 使用示例

```
目标企业：北京XX智能科技有限公司
统一社会信用代码：91110108XXXXXXXXXX
核心技术关键词：激光雷达、LIDAR、自动驾驶感知
尽调场景：招商落地
```

输入以上信息后，技能将自动执行 Step 1-7，约 3-5 分钟生成完整 HTML 尽调报告。
