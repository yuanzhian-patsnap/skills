# 智慧情报（smart-intel）Skill

## 定位

本 Skill 将"电规总院智慧情报与政策孵化平台"的全套研发经验沉淀为**可复用的自动化能力**。
只需输入目标企业名称、行业关键词，即可在数分钟内为**任意企业或研究机构**生成一个功能完整、
数据丰富、交互流畅的本地化情报门户 HTML 文件。

---

## 四大核心模块

### 1. 情报中心（Intel Center）
- 专利情报：调用 PatSnap MCP 检索近一年专利，50条/页签，纵向列表展示
- 行业新闻：调用 Web Search 获取近一年行业动态，50条
- 政策法规：调用 Web Search 检索行业政策，标注发布机构/级别/状态
- 行业研究报告：权威机构报告聚合，含 BNEF/IEA/行业协会
- 内部报告：密级标注，支持保密分类
- 功能：语义搜索、标签筛选、左侧分类树、详情弹窗、统计卡片

### 2. 政策孵化工作台（Policy Incubator）
五步全链路研究流程，每步配套图表与数据表格：
1. **需求拆解**：政策导向拆解 + 产业痛点梳理 + 优先级需求清单
2. **数据摸排**：行业统计数据 + 国内外对标经验（三省/三国）
3. **推演测算**：乐观/基准/保守三情景 + A/B/C方案比选
4. **试点路径**：试点省份 + 四阶段甘特图 + KPI监测体系
5. **成果汇总**：政策研究文稿（3000字）+ 优化建议 + 导出归档

### 3. 技术路线研判（Tech Route Analysis）
六步研判工作流，依托情报与政策孵化成果驱动：
1. **情报萃取**：机构技术布局雷达图 + 热点动态频次图
2. **政策拆解**：政策类型饼图 + 约束指标优先级图
3. **诉求映射**：6×6热力映射矩阵 + 技术需求优先级图
4. **路线推演**：TRL成熟度演进折线图 + 综合适配度雷达图
5. **落地定型**：产业化时序甘特图 + 优先级×难度象限图
6. **输出成果**：三联仪表盘（完整度/契合度/可行性）+ 导出清单
- 末尾「提交立项审查」按钮，跳转科技创新管理并自动创建项目记录

### 4. 科技创新管理（Sci-Tech Management）
14个子模块，全屏展示，每模块含统计卡片 + 详情表格 + Canvas图表：

| 模块 | 核心内容 |
|------|---------|
| 科技项目管理 | 项目全生命周期（10步）+ 在研项目清单 |
| 科技需求管理 | P1-P5优先级需求 + 5步分发流程 |
| 科技组织管理 | 院士专家 + 创新团队 + 学历结构 |
| 创新规划管理 | 十五五规划指标进度 + 专项状态 |
| 科技外协管理 | 合作机构全生命周期管理 |
| 年度计划管理 | 计划执行 + 季度/月度图表 |
| 科技平台管理 | 国家/省部/博后/院士工作站 |
| 科技奖励管理 | 获奖记录 + 申报进度追踪 |
| 科研评审管理 | 评审日历 + 专家库 + 结果分布 |
| 知识产权管理 | PatSnap真实IPC数据 + 申请趋势 |
| 高企申计管理 | 研发费用归集 + 申报评分仪表盘 |
| 科技成果管理 | TRL成熟度追踪 + 转化收益统计 |
| 标准管理 | 国标/行标/团标 + 生命周期状态 |
| 绩效管理 | 三维绩效 + TOP10排行 + 所室概览 |

## 使用方式

```bash
# 基本用法：指定目标企业和行业
python scripts/gen_smart_intel.py \
  --company "目标企业名称" \
  --industry "储能/电力/半导体/..." \
  --output portal.html

# 高级用法：接入PatSnap真实数据
python scripts/gen_smart_intel.py \
  --company "华能集团" \
  --industry "核能 风电 储能" \
  --use-patsnap \
  --use-websearch \
  --output huaneng_intel.html
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--company` | 目标企业/机构名称 | 必填 |
| `--industry` | 行业关键词（空格分隔） | 必填 |
| `--output` | 输出HTML文件路径 | `portal.html` |
| `--use-patsnap` | 是否调用PatSnap检索真实专利数据 | False |
| `--use-websearch` | 是否调用Web Search获取真实新闻/政策 | False |
| `--title` | 平台标题 | `{company} 智慧情报与政策孵化平台` |
| `--modules` | 启用模块（逗号分隔） | `intel,policy,tech,sci` |
| `--year-range` | 数据年份范围 | 近一年 |

## 数据来源策略

| 数据类型 | 来源 | MCP工具 |
|---------|------|---------|
| 专利情报 | PatSnap智慧芽 | `patsnap_search` + `patsnap_fetch` |
| 企业画像 | PatSnap企业分析 | `intellectual_property` + `technology_label` |
| 行业新闻/政策 | Web Search | `web_search` |
| 产学研合作 | PatSnap创新分析 | `industry_university_research` |
| 核心专利 | PatSnap创新分析 | `company_core_patent` |
| 行业报告 | Web Search | `web_search` |

## 输出物

1. **`portal.html`** — 完整单文件可交互情报门户（零依赖，本地直接打开）
2. **`intel_data.json`** — 情报原始数据缓存（可复用）
3. **`tech_routes.md`** — 技术路线研判报告（Markdown格式）
4. **`policy_report.md`** — 政策研究文稿（约3000字）

## 适用场景

- **企业情报分析**：快速为客户生成竞争情报门户
- **政策研究孵化**：从情报到政策方案的全链路辅助
- **科技管理数字化**：研究机构科技管理台账可视化
- **行业技术路线研判**：技术选型与攻关方向决策支持
- **招商引资/产业规划**：区域产业情报分析门户

## 技术架构

- **前端**：纯HTML5 + CSS3 + Canvas（零依赖，单文件）
- **交互**：原生JS，switchView / showSciModule / openModal 三层导航
- **图表**：Canvas 2D API，支持柱状/折线/饼图/雷达/仪表盘/甘特/象限
- **高清适配**：devicePixelRatio 动态处理，Retina屏清晰显示
- **数据注入**：Python生成时直接嵌入，运行时零网络请求

## 使用前配置
本 Skill 依赖智慧芽开放平台 MCP 服务：
- 完成安装、初次使用时需进行自检，参见 README.md
