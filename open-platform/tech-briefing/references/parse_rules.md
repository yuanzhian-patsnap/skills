# 用户输入解析规则

从用户输入中提取公司、技术主题、时间范围、地域、报告语言。

## 公司识别

1. 读取 `references/company_aliases.json`，查找公司简称
2. 命中 → 用 aliases 全称精确匹配：`ALL_AN:("全称1" OR "全称2")`
3. 未命中 → 通配符：`ALL_AN:(公司名*)`
4. **绝对不要**把公司名写入技术关键词字段（TA_ALL）

## 时间范围识别

| 格式 | 示例 | 解析结果 |
|---|---|---|
| 单年 | 2025年 | 当年 1/1 ~ 12/31 |
| 年月范围 | 2024年1月~2025年6月 | 跨月范围 |
| 年范围 | 2022-2024 | 跨年范围 |
| 相对 | 最近一年 / 近两年 / 最近半年 | 相对今天 |
| 默认 | 未指定 | 最近半年 |

日期字段选择：
- 含"申请"关键词 → 使用 `APD`（申请日）
- 否则 → 使用 `PBD`（公开日）
- 格式示例：`PBD:[20251112 TO 20260512]`

## 地域识别

| 地域 | 检索式 |
|---|---|
| 中国 / 国内 | `ANCS:(CN)` |
| 美国 | `ANCS:(US)` |
| 欧洲 | `ANCS:(EP)` |
| 日本 | `ANCS:(JP)` |
| 韩国 | `ANCS:(KR)` |
| 全球（默认） | 不加地域限制 |

## 报告语言

默认中文。用户明确指定 English / 英文则切换。

## 检索式拼装示例

用户输入："生成华为和小米5G技术简报"
- 公司：华为、小米（命中映射表）
- 技术：5G → 5G OR 毫米波 OR 大规模天线 OR 波束赋形 OR 网络切片
- 时间：默认最近半年
- 地域：全球

华为检索式：
```
ALL_AN:("华为技术有限公司" OR "HUAWEI TECHNOLOGIES CO., LTD." OR "Huawei Technologies Co., Ltd.")
AND TA_ALL:("5G" OR "毫米波" OR "大规模天线" OR "波束赋形" OR "网络切片" OR "mmWave" OR "beamforming")
AND PBD:[20251112 TO 20260512]
```
