# 关键词扩展规则

技术主题关键词必须做同义词扩展，避免漏检。

## 扩展表

| 技术主题 | 扩展关键词 |
|---|---|
| 固态电池 | 固态电池、固态电解质、聚合物电解质、硫化物电解质、全固态、solid state battery |
| 大模型 | 大语言模型、LLM、transformer、预训练、生成式AI、large language model |
| 自动驾驶 | 自动驾驶、无人驾驶、激光雷达、感知融合、路径规划、autonomous driving |
| 人形机器人 | 人形机器人、仿人机器人、双足机器人、具身机器人、机器人关节、humanoid robot |
| 5G | 5G、毫米波、大规模天线、波束赋形、网络切片、mmWave、beamforming |
| 锂电池 | 锂电池、锂离子、正极材料、负极材料、电解液、隔膜、lithium battery |
| 热管理 | 热管理、换热、冷却、温控、散热、液冷、thermal management |
| 氢能 | 氢能、燃料电池、制氢、储氢、质子交换膜、fuel cell、hydrogen |
| 手机 | 手机、智能手机、移动终端、smartphone、屏幕、摄像、充电 |

## 扩展原则

- 中英文并列，覆盖智慧芽中英文字段
- 宽泛主题（如"手机"）必须扩展出具体子方向，避免噪声
- 不在表中的主题：按专业知识给 4~8 个同义词或子方向
- 最终拼入 `TA_ALL:(kw1 OR kw2 OR ...)`，绝不和公司名混用字段

## 示例

固态电池：
```
TA_ALL:("固态电池" OR "固态电解质" OR "聚合物电解质" OR "硫化物电解质" OR "全固态" OR "solid state battery")
```
