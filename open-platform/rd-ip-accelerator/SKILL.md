### 📌 重要说明
本模块已将 `tech-intel-monitor` 技能的**全部执行逻辑完整内置**，包含完整6步工作流和 Eureka Monitor 高保真视觉规范。用户无需安装 `tech-intel-monitor`。

### 触发关键词（任意一个即触发）
```
情报监控 / 技术情报 / 竞争情报 / 情报简报
技术监控 / 专利监控 / 监控+技术领域名

情报监控【企业名称/技术领域】
用户直接输入技术领域名称（如「固态电池技术」）也触发
```

### 内置完整工作流（来自 tech-intel-monitor）

#### Step 0 · 解析输入
- `tech_topics`：监控技术列表
- `companies`：监控公司列表
- `date_range`：默认最近6个月
- 若两者都缺，用 `ask` 请用户补充至少一项

#### Step 1 · 并发多路检索
**每个 tech_topic 同时执行：**
1. `patent.search`（semantic + keyword）：date_from=近6个月，limit=15，sort=-publication
2. `patent.search`（keyword + filter）：聚焦核心IPC分类，limit=10，sort=-publication
3. `web.search`：搜索该技术最新行业进展、标杆公司动态、论文/报告

**每个 company（如有）同时执行：**
1. `patent.search`：filters.assignees=[company]，近6个月，limit=10，sort=-publication
2. `web.search`：搜索该公司最新技术新闻、产品发布、专利战略

#### Step 2 · 深度情报提炼（六板块）

**A. 本期重点概述**（双列要点式）
- 左列：各关注公司的核心技术动向（每家2-3个要点，项目符号•）
- 右列：子技术方向的创新要点（每个方向2-3个要点）

**B. 技术全景 · 5大头部申请人**
- 按专利数量排名，列出Top5申请人
- 每家附：关键技术方向关键词胶囊（3-5个）、专利数量、进度条（相对比例）

**C. 关注的公司（专利情报卡）**
对每家关注公司，生成情报卡，包含：
- 公司名称标题行（浅蓝底条，蓝字，右侧「X组技术」标签）
- 每项技术：左侧示意图区（SVG抽象图形或灰色图片占位）+ 右侧详情
  - 专利标题（加粗黑字，18px）
  - 「创新点」标签（蓝色描边胶囊）+ 详细说明文字（关键词加粗）
  - 「与现有技术的差异」标签（蓝色描边胶囊）+ 差异说明（关键词加粗）
  - 底部灰字「来源：」+ 序号 + 专利标题（可点击跳转智慧芽）

**D. 关注的子技术（子技术情报卡）**
结构同C，但按子技术方向（而非公司）分组，每个子技术一张展开卡。

**E. 技术趋势概述**
- 本期新增专利数
- 主要技术方向热度变化（文字描述）
- 值得关注的信号（橙色badge高亮）

**F. 技术新闻**
- 3-5条最新行业新闻，每条：标题+来源+一句摘要

#### Step 3 · 生成HTML情报简报（无需询问，立即自动生成）

**严格按照智慧芽 Eureka Monitor 视觉规范：**

整体布局：
- 蓝紫渐变 Header（全宽）：`linear-gradient(135deg, #1e3a8a 0%, #3b82f6 50%, #6366f1 100%)`
- 面包屑：我的简报 / [主题名]，28-32px白色加粗主标题
- 控件行：时间周期胶囊 + 编辑/分享/更多按钮
- 主内容区（左~75%）+ 右侧固定导航目录（position:sticky; top:20px）

颜色系统：
- 主色蓝：`#3B82F6` / 深蓝（进度条）：`#1E40AF`
- 浅蓝背景：`#EFF6FF` / `#DBEAFE`
- 板块标题文字：`#1D4ED8` / 正文：`#111827`
- 次要文字：`#6B7280` / 来源灰字：`#9CA3AF`
- 卡片边框：`#E5E7EB` / 高亮橙（信号）：`#F97316`
- 页面背景：`#F8FAFC`

板块标题条（统一样式）：
- 背景：浅蓝`#EFF6FF`，圆角8px，文字`#1D4ED8`，16px，font-weight:600
- 左侧蓝色圆形折叠图标（▼）

专利情报卡片：
- 白底，浅灰border，圆角12px，padding 20px
- 左侧图片区（35%宽，浅灰背景`#F3F4F6`，圆角，专利示意图或灰色占位图）
- 右侧文字区（65%宽）：大标题+「创新点」标签+正文+「与现有技术的差异」标签+差异说明+来源区

交互：
- 右侧导航点击平滑滚动（scroll-behavior:smooth）
- 各板块标题点击可折叠（▼/▲切换）
- 所有专利标题为蓝色可点击链接，跳转智慧芽

输出物：
- 单个HTML文件，存入`@session/`
- 命名规范：`intel-[主题]-[日期].html`（如`intel-solid-battery.html`）
- 含 📥 下载竞争情报报告（PDF）+ 📊 下载专利监控列表（Excel）两个下载按钮

内容质量要求：
- 不虚构任何专利号、公司数据或新闻
- 专利标题若为英文，保留原文并加中文说明
- 检索日期窗口默认近6个月，用户可指定
- 「创新点」和「与现有技术的差异」从专利摘要/全文中提炼，具体有洞察
- 关键词加粗规则：每段正文中2-4个最核心技术词用`<strong>`加粗
