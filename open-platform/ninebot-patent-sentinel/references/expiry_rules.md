## 五级预警定义

| 等级 | 符号 | 条件 | 行动要求 |
|------|------|------|--------|
| 已到期 | ⚫ | 剩余天数 ≤ 0 | 确认公有领域，排查竞对模仿 |
| 紧急 | 🔴 | 0 < 剩余天数 ≤ 180 | 本月内完成续期决策 |
| 预警 | 🟠 | 180 < 剩余天数 ≤ 365 | 本季度内完成续期决策 |
| 关注 | 🟡 | 365 < 剩余天数 ≤ 730 | 年度专利资产盘点时决策 |
| 安全 | ✅ | 剩余天数 > 730 | 正常维护 |

## 续期决策框架

### P1 立即行动（6个月内到期）
- 评估技术有效性：该专利保护的技术是否仍在产品中使用？
- 评估商业价值：是否有竞对已在使用或可能使用该技术？
- 评估维持费用：续期费用 vs 技术价值
- 法律操作：向国家知识产权局缴纳年费（可附加滞纳金宽限期6个月）

### P2 本季度决策（6-12个月内到期）
- 启动内部技术有效性评估
- 检索竞对是否已有相同/相似专利布局
- 制定续期or放弃决策备忘录

### P3 长期规划（1-2年内到期）
- 纳入年度专利资产盘点
- 关注相关技术域的后续申请情况
- 评估是否需要申请后续改进专利
