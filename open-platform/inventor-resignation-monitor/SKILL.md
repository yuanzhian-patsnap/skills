# 发明人离职监控简报 Skill

## 功能概述

面向企业IP团队，自动完成以下分析流程并输出HTML简报：

1. **识别疑似离职发明人**：检索目标公司历史专利发明人，筛选"近N年无新申请"的人员
2. **监控他处专利申请**：在监控窗口期内，检索这些发明人在其他公司/机构的专利申请
3. **技术相似度分析**：将新申请专利与原公司技术领域比对，给出风险等级
4. **输出HTML简报**：包含风险等级（🔴高/🟠中/🟡持续关注/🟢低）、专利对比表、预警建议

## 输入参数

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `company_name` | string | 目标公司名称（必填，与发明人二选一） | 浙江孚邦 |
| `tech_domain` | string | 技术领域关键词（可选，提升检索精准度） | 外骨骼、康复机器人 |
| `inventor_names` | list[str] | 直接指定发明人姓名列表（可选，与公司二选一） | ["陈伟海", "王业伟"] |
| `inactive_years` | int | 认定离职的无申请年限（默认5年） | 5 |
| `monitor_years` | int | 监控窗口年数（默认2年） | 2 |
| `output_lang` | str | 报告语言（默认zh） | zh |

## 输出

- `@session/reports/inventor_monitor_<公司名>_<日期>.html` — HTML格式简报
- 终端摘要：高风险发明人列表、命中专利数量、风险等级汇总

## 分析方法论

### Step 1：获取历史发明人
- 检索目标公司全部专利（按发明人字段聚合）
- 统计每位发明人最近一次在该公司申请专利的年份
- 筛选条件：最后申请年 < 当前年份 - inactive_years → 标记为"疑似离职"

### Step 2：离职发明人他处专利监控
- 对每位疑似离职发明人，以发明人姓名 + 技术领域关键词检索
- 过滤条件：申请日在监控窗口内 AND 申请人不含目标公司名称
- 收集命中专利列表

### Step 3：技术相似度评估
- 高风险：新申请专利技术与原公司核心技术高度相关（同IPC小类/相同技术词）
- 中风险：技术有一定关联（同IPC大类/部分技术词重叠）
- 低风险：技术无明显关联

### Step 4：生成HTML简报
- 风险汇总表（颜色编码）
- 每位高/中风险人员的专利对比详情
- 建议行动项（法务比对/持续监控/暂无风险）

## 使用示例

```
# 按公司+技术领域模式
公司：浙江孚邦科技有限公司
技术领域：外骨骼机器人、康复设备
监控窗口：近2年

# 按指定发明人模式
发明人：陈伟海
技术领域：外骨骼、肌电信号、下肢康复
监控窗口：近2年
```

## 注意事项

- 专利有18个月公开时滞，最近18个月内申请的专利可能尚未公开，实际风险可能被低估
- 同名发明人可能存在误报，建议结合IPC分类和技术词二次确认
- 建议每季度执行一次滚动监控
