# 示例数据说明

由于专利数据受版权保护，本 skill 不提供真实专利数据文件。

## Excel 数据格式要求

你需要准备一个包含专利清单的 Excel 文件（.xlsx 格式），包含以下列：

### 必需列

| 列名 | 说明 | 示例 |
|------|------|------|
| `公开(公告)号` | 专利公开号或公告号 | US20260114661A1, CN121890870A |
| `标题` | 专利标题 | 一种咖啡研磨机的控制方法 |
| `[标]当前申请(专利权)人` | 申请人或专利权人 | 德隆吉器具有限公司 |
| `简单法律状态` | 法律状态 | 有效, 审中, 失效 |

### 推荐列（用于提升质量）

| 列名 | 说明 | 用途 |
|------|------|------|
| `Patsnap专利标题` | AI生成的精简标题 | 用于简报展示 |
| `技术问题` | 专利解决的技术问题 | 用于关键词匹配 |
| `技术手段` | 采用的技术手段 | 用于关键词匹配 |
| `技术功效` | 技术效果描述 | 用于关键词匹配 |
| `申请日` | 申请日期 | 用于统计分析 |
| `公开日` | 公开日期 | 用于统计分析 |

### 可选增强内容

**摘要附图：**
- 在 Excel C 列嵌入专利摘要附图（图片格式）
- 生成的 HTML 会自动提取并嵌入

**专利链接：**
- 在 B 列（公开公告号）添加超链接
- 链接到 Patsnap 或其他专利数据库
- 生成的 HTML 会保留可点击链接

## 数据来源

### 推荐来源：Patsnap 专利数据库

1. 登录 [Patsnap](https://analytics.zhihuiya.com/)
2. 使用专业检索式搜索专利
3. 选择相关专利
4. 导出为 Excel 格式，包含以下字段：
   - 公开(公告)号
   - 标题
   - [标]当前申请(专利权)人
   - 简单法律状态
   - Patsnap专利标题
   - 技术问题
   - 技术手段
   - 技术功效
   - 摘要附图（勾选"包含图片"选项）

### 其他数据源

也可以使用其他专利数据库导出的 Excel，但需要：
1. 确保包含必需的4个列
2. 列名可能需要手动调整为上述标准列名
3. 或修改脚本中的列名映射

## 数据规模建议

- **小型测试**：10-50 件专利
- **正常分析**：100-500 件专利
- **大型分析**：500-2000 件专利

**注意：** 专利数量过多（超过2000件）会导致：
- 标引时间变长
- HTML 文件体积过大（如果包含大量图片）
- 建议分批处理或筛选后再分析

## 数据质量建议

### 1. 时间范围筛选
在数据库中先按时间范围筛选专利，确保数据聚焦：
```
APD:[20260101 TO 20260430]  # 申请日范围
PBD:[20260101 TO 20260430]  # 公开日范围
```

### 2. 技术领域筛选
使用 IPC 分类号或关键词初步筛选：
```
IPC:(A47J31/*)  # 咖啡机相关IPC分类
TAC:(咖啡机 OR coffee machine)
```

### 3. 去重
- 确保没有重复的专利号
- 如果同一专利有多个公开号，选择最新的

### 4. 数据清洗
- 删除明显不相关的专利
- 补全缺失的必需字段
- 统一法律状态用词（有效/审中/失效）

## 常见问题

### Q: Excel 中没有"Patsnap专利标题"列怎么办？

A: 没关系，这是可选列。如果没有，HTML 简报中会显示"-"或原始标题。

### Q: 如何在 Excel 中添加专利链接？

A: 
1. 选中 B 列的专利号单元格
2. 右键 → 插入超链接
3. 输入 Patsnap 链接（如 `https://analytics.zhihuiya.com/patent/detail/...`）
4. 确定

### Q: 如何在 Excel 中嵌入摘要附图？

A: 
1. 从 Patsnap 下载专利摘要附图
2. 在 Excel C 列对应行右键 → 插入 → 图片
3. 调整图片大小（建议 200x150 像素）

### Q: 可以使用 CSV 格式吗？

A: 不推荐。CSV 格式：
- 不支持嵌入图片
- 不支持超链接
- 可能出现编码问题

如果只有 CSV，请先转换为 Excel：
```python
import pandas as pd
df = pd.read_csv('data.csv', encoding='utf-8')
df.to_excel('data.xlsx', index=False)
```
