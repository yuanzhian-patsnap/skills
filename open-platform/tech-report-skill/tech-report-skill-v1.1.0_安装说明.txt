tech-report-skill v1.1.0 安装说明
========================================

## 方法 1：使用 tar.gz 安装

tar -xzf tech-report-skill-v1.1.0.tar.gz
cd tech-report-skill-v1.1.0
bash install.sh

## 方法 2：使用 zip 安装（如果有）

unzip tech-report-skill-v1.1.0.zip
cd tech-report-skill-v1.1.0
bash install.sh

## 方法 3：手动安装

1. 解压文件到任意位置
2. 运行：bash install.sh

## 安装后使用

在 Claude Code 中输入：
"请用XX技术主题生成研发简报"

或命令行运行：
cd ~/.claude/skills/tech-report-skill
bash scripts/run.sh <Excel路径> <技术主题> <开始日期> <结束日期>
