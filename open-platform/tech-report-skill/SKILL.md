# 技术研发简报生成 Skill

## 用途

基于专利Excel数据，自动生成美观的技术研发简报HTML文件，包含：
- 行业动态
- 重点公司技术动向
- 技术分类与专利解读
- 专利详情（含摘要附图）

## 工作流程

1. **关键词配置**：根据技术主题定义包含/排除关键词
2. **专利标引**：自动标记高相关专利
3. **数据分析**：提取公司、技术分类、行业动态
4. **HTML生成**：生成响应式、可交互的简报页面

## 使用方法

### 快速开始

```bash
cd ~/.claude/skills/tech-report-skill
bash scripts/run.sh "<Excel文件路径>" "<技术主题>" "<开始日期>" "<结束日期>"
```

**参数说明：**

- `<Excel文件路径>`：包含专利清单的Excel文件（必填）
- `<技术主题>`：技术领域名称，如"BIPV"、"固态电池"（必填）
- `<开始日期>`：报告起始日期，格式YYYYMMDD（必填）
- `<结束日期>`：报告结束日期，格式YYYYMMDD（必填）

**示例：**

```bash
bash scripts/run.sh "/path/to/BIPV专利清单202604.XLSX" "BIPV" "20260201" "20260430"

bash scripts/run.sh "/path/to/固态电池专利.xlsx" "固态电池" "20260101" "20260331"
```

## 输入文件要求

Excel文件需包含以下列（列名可能略有不同）：

**必需列：**
- `公开(公告)号`：专利号
- `标题`：专利标题
- `[标]当前申请(专利权)人`：申请人/公司名称
- `简单法律状态`：有效/审中/失效等

**可选列（用于关键词匹配）：**
- `Patsnap专利标题`
- `技术问题`
- `技术手段`
- `技术功效`

**可选列（用于展示）：**
- `申请日`
- `公开日`
- `技术分类`
- `技术相关性`

## 输出文件

生成的文件保存在与输入Excel相同的目录：

1. **标引文件**：`{技术主题}专利清单_{日期范围}_标引.xlsx`
   - 新增"相关性标引"列，高相关专利高亮显示

2. **HTML简报**：`{技术主题}_report_{日期范围}.html`
   - 响应式设计，支持移动端
   - 导航菜单，快速跳转
   - 专利详情含摘要附图
   - 可点击专利号跳转Patsnap

## 配置文件

### 1. 关键词配置文件

创建 `~/.claude/skills/tech-report-skill/config/{技术主题}_keywords.py`

```python

    "曲面光伏", "光伏瓦", "户用BIPV", "光储一体",
    "curved solar tile", "solar roof tile", "BIPV energy storage",

# 排除关键词（命中则降级）
EXCLUDE_KEYWORDS = [
    "光伏幕墙", "光伏外立面", "平屋顶光伏",
    "solar facade", "PV curtain wall", "flat roof PV",

# 公司简称映射
COMPANY_SHORT = {
    '特斯拉公司(美国)': '特斯拉',
    '隆基绿能科技股份有限公司': '隆基绿能',

```

## 常见问题

**Q: Excel文件中没有"技术相关性"列怎么办？**

A: 脚本会自动运行标引流程，基于关键词匹配生成"相关性标引"列。

**Q: 如何修改关键词？**

A: 编辑 `config/{技术主题}_keywords.py` 文件，修改 `INCLUDE_KEYWORDS` 和 `EXCLUDE_KEYWORDS` 列表。

**Q: 生成的HTML没有摘要附图？**

A: 确保Excel文件的B列（公开公告号列）包含图片，或者Excel文件中有图片嵌入。

**Q: 如何自定义公司技术总结？**

A: 编辑 `config/{技术主题}_content.py` 文件中的 `COMPANY_DATA` 列表。

## 核心特性

### 1. 数据完整性保障
- ✅ **保留Excel超链接**：专利公开号可直接点击跳转到Patsnap原文
- ✅ **保留专利附图**：自动提取Excel内嵌图片并转为Base64嵌入HTML
- ✅ **完全自包含**：所有图片、样式内联，可离线查看或邮件发送

### 2. 智能专利分类
- ✅ **关键词匹配算法**：基于关键词相关度自动为每个技术分类挑选Top-20专利
- ✅ **相关度评分**：统计匹配关键词数量，优先展示最相关专利
- ✅ **动态分类**：无需手动配置所有专利号，系统自动分类

### 3. 美观的视觉设计
- ✅ **装饰性背景图**：Hero区域自动下载技术主题相关图片作为背景
- ✅ **响应式布局**：支持桌面端和移动端，卡片式设计清晰美观
- ✅ **交互导航**：固定导航栏，支持快速跳转到任意章节

### 4. 可点击的内容链接
- ✅ **新闻链接**：每条新闻都有"查看原文"按钮跳转到新闻网页
- ✅ **专利链接**：所有专利号都可点击跳转到Patsnap专利详情页

## 常见问题

**Q: 为什么专利链接和图片能保留？**

A: 脚本使用 `openpyxl` 直接操作Excel文件，在原文件上添加标引列，完整保留所有超链接和内嵌图片。详见 [IMPROVEMENTS.md](./IMPROVEMENTS.md)。

**Q: 如何确保每个技术分类有足够的专利？**

A: 系统会根据关键词匹配度自动为每个技术分类选取Top-20专利。你只需在 `_keywords.py` 中定义技术分类的关键词列表即可。

**Q: 简报中的背景图片会丢失吗？**

A: 不会。背景图片在生成时下载并转为Base64编码嵌入HTML，简报发送或复制后图片依然存在。

**Q: 如何让新闻标题可以跳转？**

A: 在 `{技术主题}_content.py` 的 NEWS 配置中，每条新闻的第5个参数填写新闻链接URL。生成的简报会自动添加"查看原文"按钮。
