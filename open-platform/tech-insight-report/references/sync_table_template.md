# 跨章节数据同步表模板

> 使用说明：在Phase 0开写前填写此表。修改任意一处数据时，同步表对应行的所有列均需同步更新。

---

## 使用方法

1. 将本模板复制到工作目录，重命名为 `sync_table_[技术主题].md`
2. Phase 0阶段填入所有计划引用的关键数据
3. 每次修改报告时，先更新同步表，再修改HTML
