# 示例问题

## 公司级资产盘活

```text
使用 $sleeping-patent-asset-activation，以「XX 公司」为对象，盘点其沉睡专利资产，识别最值得盘活的 Top 20 专利，并生成客户可读的 HTML 报告；如果环境支持，同时导出 PDF。报告需包含资产分层、推荐盘活路径和管理层摘要。
```

## 专利清单快速体检

```text
使用 $sleeping-patent-asset-activation，分析我上传的专利清单，判断哪些专利适合保留、许可、转让或放弃，并生成 HTML 体检报告，包含每件专利的价值评分、法律状态风险和商业化建议。
```

## 年费续缴决策

```text
使用 $sleeping-patent-asset-activation，帮我分析这批即将缴纳年费的专利，生成续缴决策 HTML 报告，判断哪些值得继续维持、哪些可以考虑放弃，并说明每项建议背后的技术价值、市场价值和交易洁净度依据。
```

## 对外许可机会识别

```text
使用 $sleeping-patent-asset-activation，从我司专利组合中找出适合对外许可的专利，生成一份可给商务团队使用的 HTML 机会清单，按潜在买方行业、许可吸引力、谈判优先级排序。
```

## 专利转让包设计

```text
使用 $sleeping-patent-asset-activation，基于我司在「智能座舱 / 电池管理 / 工业传感器」方向的专利，设计 3 个可对外转让的专利包，并生成 HTML 报告；每个包说明核心技术、目标买方、估值区间和交易风险。
```

## 研发复用视角

```text
使用 $sleeping-patent-asset-activation，分析我司沉睡专利中是否有可被当前「XX 新产品线」复用的技术资产，并生成研发复用 HTML 报告，输出可复用专利、对应产品场景、复用难度和建议对接部门。
```

## 买方推荐

```text
使用 $sleeping-patent-asset-activation，针对这 10 件高价值但未实施的专利，生成买方推荐 HTML 报告，识别潜在买家或被许可方，说明推荐理由、买方可能需求和初步触达话术。
```

## 交易洁净度审查

```text
使用 $sleeping-patent-asset-activation，对这批拟挂牌交易的专利做交易前审查，并生成交易洁净度 HTML 报告，重点检查法律状态、同族、权利要求稳定性、剩余保护期和可能影响成交的风险。
```

## 管理层汇报版

```text
使用 $sleeping-patent-asset-activation，把我司沉睡专利盘活分析整理成一页式 HTML 管理层报告，突出资产规模、可盘活金额、优先行动、风险提示和 90 天推进计划。
```

## 客户可交付成果物

```text
使用 $sleeping-patent-asset-activation，基于本次分析生成一份可直接发给客户的 HTML 或 PDF 报告，风格采用瑞士国际主义与极简科技风，内容包括资产分层、重点专利卡片、商业化路径和行动清单。
```
